React Notes App

This is a simple note-taking app built with React. It lets users add and delete notes in real time using component state. The app uses the useState hook to keep track of notes and updates the UI automatically as changes happen. It’s built with a clean, modular structure: the App component handles state, CreateArea manages the input form, and each Note component receives props to handle deleting. Notes are added and removed using React best practices like functional state updates and array filtering. Data flows one way through the app using prop drilling to keep things consistent and easy to manage.

Flight Dashboard Project

The project showcases a flight dashboard with a dynamic, 2D game-inspired interface, utilizing real-time location tracking through Leaflet. Users interact by selecting two checkpoints (represented by planes) to view available flight options. These checkpoints help determine the user’s journey, enabling geographical calculations like flight path distance. The code processes a JSON file containing airport data, converting geographic coordinates from degrees and minutes to decimal format using regex, and places markers for each international airport. When a marker is clicked, it fetches and displays weather data for that airport. The interface also allows users to select two airports, calculate the distance between them, and draw a line on the map to indicate the flight route. Flight options can be filtered by criteria such as duration, cost, and plane type. Users can add selected flights to a cart, view detailed booking information in a modal, and submit bookings through an integrated form.
