Flight Dashboard Project

The project features a flight dashboard with a dynamic 2D game-inspired interface. It integrates real-time location tracking using Leaflet, where users must click on two checkpoints (represented by sunflowers) to display available flight options. These checkpoints help determine the user's journey, allowing for geographical calculations related to the flight path. Once the checkpoints are selected, the flight options are filtered based on criteria like duration, cost, and plane type. The dashboard allows users to add flights to a cart, view booking details in a modal, and submit their booking through a form. Data is processed in JSON format, and the interface combines map visualization, dynamic updates, and interactive features for a seamless user experience.

Interactive Quiz

This project is an interactive web-based quiz application. Users enter their name to begin the quiz, where questions are displayed as animated cards. The application uses jQuery for real-time DOM updates and Bootstrap for responsive design. A timer tracks the quiz duration, and scores are calculated based on the user’s responses. Results, including the score and time taken, are shown in a modal. The quiz incorporates smooth animations, sequential question loading, and optional hints for a user-friendly experience.

Interactive User Management System

This project is a user management system using HTML, CSS, and JavaScript ES6. It features a login modal where users enter their username, validated via regex. Once logged in, users are presented with a personalized dashboard showing their profile and information. Admins can view and delete non-admin users, while non-admins can only see admins and their own details. The dashboard includes dynamic animations for user actions and a responsive design. Key features include user validation, dynamic card creation, and a delete function with an animated slide-out effect.

Networking Project

I developed a real-time temperature monitoring system using Raspberry Pi, CentOS, and Python. The Pi reads temperature data via a 1-Wire sensor and is secured with a firewall (ufw) allowing future access on port 8000. I set up Apache2 to display the temperature readings on a web page and modified the web_temperature.py program to control LEDs based on temperature changes and track the data over time.

Counting Project with Python and Raspberry Pi

In this project, I used Raspberry Pi, CentOS, and Python to create programs for Pi Hat control. I started by testing the Hat LEDs, writing Python scripts with the RPi.GPIO library to control the yellow, red, and green LEDs through specific GPIO pins. I then developed a "counting.py" script that changed the LED color based on button presses—red for odd numbers, green for even numbers, and yellow for powers of 2.
